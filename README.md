# 🎓 Form-and-Tables

> **Primera instancia evaluatoria** del curso de **Python + FastAPI** en el ITU

## 📌 Descripción

Este proyecto permite, a partir de un **formulario adaptativo**, añadir información de manera cómoda a una **tabla de Excel** y generar **actas escolares** para estudiantes de nivel secundario.

La aplicación determina automáticamente las materias que los alumnos deberán **cursar** o **rendir**, en función de su situación académica.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Python `3.13.0`
- **Dependencias**:
  - `tkinter` – Para la interfaz gráfica interactiva 🖼️
  - `openpyxl` – Para el manejo de archivos Excel 📊

## ✅ Requisitos cumplidos según las consignas de la instancia de evaluación

- **Menú con funciones CRUD**  
  📋 Se incluye un menú accesible desde la interfaz para gestionar altas, bajas, modificaciones y consultas de datos.

- **Interfaz gráfica interactiva**  
  🧩 Desarrollada con `tkinter`, permite el ingreso de datos mediante formularios intuitivos.

- **Persistencia con archivo JSON**  
  💾 Los cambios realizados en las tablas se guardan utilizando archivos `.json`, asegurando la persistencia de la información.

- **POO – Clases y Objetos**  
  👨‍🏫 Se implementan clases para representar **alumnos**, **materias**, y sus relaciones, siguiendo los principios de la Programación Orientada a Objetos.

- **Uso de librerías de terceros**  
  📦 Se integran `tkinter` y `openpyxl` como dependencias externas.
